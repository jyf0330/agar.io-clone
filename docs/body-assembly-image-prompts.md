# Body Assembly Image Prompts

这些提示语用于生成《神的诞生》开局身体零件资源。所有图片都应导出为 512x512 透明背景 PNG，零件居中，方便前端按固定锚点叠加。

## 通用负面提示

不要写实，不要恐怖血腥，不要赛博科幻，不要现代 UI 质感，不要复杂背景，不要强透视，不要复杂阴影，不要把其他身体部位画进来，不要文字，不要水印。
