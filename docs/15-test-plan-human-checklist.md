# 真人测试清单

这份给真人测试员使用。重点不是“代码有没有返回字段”，而是玩家是否看得懂、玩得顺、反馈是否自然。

每项记录：通过/失败/不确定、截图或录屏、复现步骤、体感备注。

## 1. 首屏与设置

- [ ] `真人优先` `H01-01` 首屏能看懂：标题、昵称、录制同意、开始、观战、绘制名片、设置、玩法说明。
- [ ] `真人优先` `H01-02` 输入昵称并点击开始，流程自然，不知道下一步时不会迷路。
- [ ] `真人优先` `H01-03` 切换 English / 中文后，主要文案自然，没有明显漏翻或错位。
- [ ] `真人优先` `H01-04` 打开设置，理解“显示边界”“显示质量”“离屏继续移动”“圆形食物”“深色模式”的含义。
- [ ] `真人优先` `H01-05` 取消“同意录制本局用于历史回响”后，玩家能理解这个选择会影响历史回响。

## 2. 身体签名 V5

- [ ] `真人优先` `H02-01` 第一次开始游戏时弹出签名面板，缺失部位、三张参考图、画布、清空、提交、跳过都清楚。
- [ ] `真人优先` `H02-02` 不画直接提交时，提示“先画几笔再提交”，文案清楚。
- [ ] `真人优先` `H02-03` 画几笔后提交，档位反馈能理解：无回声 / 微弱回声 / 强回声。
- [ ] `真人优先` `H02-04` 画得接近某张参考图时，结果看起来可信，不显得随机。
- [ ] `真人优先` `H02-05` 清空、重画、跳过三个动作都符合预期。
- [ ] `真人优先` `H02-06` 手机上触摸画签名可用，不误滚页面，不遮挡按钮。

## 3. 名片、头像草稿与画板

- [ ] `真人优先` `H03-01` 点击“绘制名片”能打开画板，默认骨架背景加载，圆形预览区域清晰。
- [ ] `真人优先` `H03-02` 画笔、颜色、粗细滑杆容易使用。
- [ ] `真人优先` `H03-03` 矩形、圆形、直线、文字工具插入后，用户知道如何继续编辑。
- [ ] `真人优先` `H03-04` 选择、移动、橡皮、撤销、重做、清空符合普通用户预期。
- [ ] `真人优先` `H03-05` 保存后开始菜单预览和游戏内我的名片 HUD 让人知道保存成功。
- [ ] `真人优先` `H03-06` 新建图片、保存到草稿、历史草稿容易理解。
- [ ] `真人优先` `H03-07` 高级图层 Base / Eyes / Hair 的切换、显示/隐藏、锁定/解锁能被理解。
- [ ] `真人优先` `H03-08` 放大/缩小、滚轮缩放、中键拖动、键盘方向平移不会让用户迷失。
- [ ] `真人优先` `H03-09` 两个玩家靠近后，目标名片 HUD 和 cell 内头像能帮助识别对方。
- [ ] `真人优先` `H03-10` 如果绘图工具加载慢或失败，提示能被理解，不像页面坏掉。

## 4. 移动与基础玩法手感

- [ ] `真人优先` `H04-01` 鼠标移动方向自然，以屏幕中心为原点，离中心越远越明显。
- [ ] `真人优先` `H04-02` 方向键控制上下左右，松开按键停止对应方向。
- [ ] `真人优先` `H04-03` 鼠标移出画布时，“离屏继续移动”关闭则停止，开启则继续。
- [ ] `真人优先` `H04-04` 手机触摸移动可用，触摸不会导致页面乱滚。
- [ ] `真人优先` `H04-05` 显示边界开启时能看到地图边界，关闭后不干扰视野。
- [ ] `真人优先` `H04-06` 吃小食物会变大，质量显示开启时数字增长，反馈明确。
- [ ] `真人优先` `H04-07` 质量越大移动越慢，手感变化可感知。
- [ ] `真人优先` `H04-08` 按 `W` 或手机 feed 按钮喷射质量，方向和反馈自然。
- [ ] `真人优先` `H04-09` 按空格或手机 split 按钮分裂，声音和动作不突兀。
- [ ] `真人优先` `H04-10` 吃病毒会触发分裂，玩家能看出这不是普通加质量。

## 5. 双人、多人与社交

- [ ] `真人优先` `H05-01` 两个玩家同时进入，双方能互相看见，位置同步可接受。
- [ ] `真人优先` `H05-02` 大玩家完整包住小玩家时吞噬成功；边缘碰撞不应让人感觉误吞。
- [ ] `真人优先` `H05-03` 被吞玩家看到死亡/结算，能理解发生了什么。
- [ ] `真人优先` `H05-04` 排行榜能看懂，当前玩家高亮明显。
- [ ] `真人优先` `H05-05` 按 Enter 聚焦聊天，输入普通消息并回车，其他玩家可见。
- [ ] `真人优先` `H05-06` 聊天输入打开、Esc 返回画布、继续游戏的过程不别扭。
- [ ] `真人优先` `H05-07` 观战模式能看全图，且不像自己正在参与战斗。

## 6. 连接、关系、实体化、身体

- [ ] `混合` `H06-01` 两个玩家靠近后按 `E`，连接过程反馈明显，玩家知道自己在连接谁。
- [ ] `混合` `H06-02` 成功共鸣和失败断裂的差异能被感知，不只是 HUD 数字变化。
- [ ] `混合` `H06-03` intimacy / spike / pollution 的中文语义和显示位置能看懂。
- [ ] `混合` `H06-04` 有额外 HAND 后，连接距离变远这件事能被实际感知。
- [ ] `混合` `H06-05` HUD 显示实体化值与 HOLLOW / PARTIAL / REAL / OVERREAL 阶段，玩家知道它在表达成长。
- [ ] `混合` `H06-06` 地图部位图标/颜色/位置能被注意到。
- [ ] `混合` `H06-07` 拾取部位后，身体 HUD 部位数增加，玩家知道自己变强了。
- [ ] `混合` `H06-08` 吃掉玩家后掠夺部位，结算能说明部位来源。

## 7. NPC、跟宠、任务与记忆

- [ ] `真人优先` `H07-01` 入场后能看到 NPC 或跟宠相关反馈：Mochi、Doudou、Wugui。
- [ ] `真人优先` `H07-02` NPC 气泡显示位置正确，不遮挡主要操作，文字停留时间合适。
- [ ] `真人优先` `H07-03` 跟宠 HUD 显示当前跟宠名称与人格，玩家能理解谁在跟着自己。
- [ ] `真人优先` `H07-04` NPC 聊天输入可见，快捷问题、任务、结算、宠物切换按钮可点击。
- [ ] `真人优先` `H07-05` 切换宠物后，反馈“现在我跟着你。”自然、不突兀。
- [ ] `真人优先` `H07-06` 输入“哪里有回声？”“附近有什么部位？”等问题，NPC 回应有帮助。
- [ ] `真人优先` `H07-07` 输入“任务”，靠近 NPC 时生成奖励部位；离太远时提示先靠近，提示能理解。
- [ ] `真人优先` `H07-08` NPC 回答符合角色感：安静诗意 / 调皮 / 稳重，不机械、不太长、不频繁刷屏。
- [ ] `真人优先` `H07-09` 第二局回来时，NPC 提到上一局记忆或期待；无历史时不胡乱冒充记忆。

## 8. 历史回响与结算体验

- [ ] `真人优先` `H08-01` demo mode 下出现历史回响 ghost 时，移动轨迹像过去玩家，不像普通玩家或静态图标。
- [ ] `真人优先` `H08-02` 回响聊天/部位锚点触发时，气泡或部位出现位置合理。
- [ ] `真人优先` `H08-03` 取消录制同意后，结算说明本局未写入历史，用户能理解。
- [ ] `真人优先` `H08-04` 回合时间到期或快速结算后，结算面板清楚说明为什么结束、赢者是谁、拿了哪些部位。
- [ ] `真人优先` `H08-05` 宠物总结自然，像一局结束后的反馈，而不是调试日志。

- [ ] `真人优先` `H09-01` 网格、食物、病毒、玩家、部位、ghost、pet 视觉层级清楚，互相不混淆。
- [ ] `真人优先` `H09-02` 深色模式下文字、网格、食物、HUD 对比度足够。
- [ ] `真人优先` `H09-03` split 声音、spawn 声音不突兀，不因浏览器自动播放策略造成明显错误。
- [ ] `真人优先` `H09-04` 手机上开始菜单、签名、名片、游戏 HUD 不互相遮挡。
- [ ] `真人优先` `H09-05` 手机 split/feed 按钮位置能按到，不挡住关键视野。
- [ ] `真人优先` `H09-06` 长昵称、空排行榜、无名玩家、无名片玩家时 UI 不破版。
